\cvevent{Autonomous Mobile Robot}{IMERIR}{2023}{}
Engineered a hybrid navigation system combining classical robotics with deep learning control.
\begin{itemize}
\item Orchestrated the \textbf{ROS Navigation Stack}, utilizing \textbf{SLAM} for mapping and \textbf{AMCL} for probabilistic localization.
\item Integrated a \textbf{CNN model} (trained on MNIST) to drive navigation decisions based on real-time visual marker recognition.
\end{itemize}
