\cvevent{Swarm Intelligence Exploration}{Université Claude Bernard Lyon 1}{2024}{}
Developed a simulation of autonomous robots collecting resources using \textbf{Ant Colony Optimization} (Pheromones).
\begin{itemize}
\item Programmed decentralized logic where agents deposit digital pheromones to optimize retrieval paths.
\item Solved the "coordination problem" without a central server, mimicking biological swarm behavior.
\end{itemize}
