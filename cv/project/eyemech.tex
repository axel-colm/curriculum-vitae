\cvevent{EyeMech}{Personal}{2021}{}
Built a real-time vision and control system for mechatronic eyes (based on Will Cogley’s design).
\begin{itemize}
\item Integrated cameras for visual tracking and person-following.
\item Implemented smooth motion control with servo feedback and OpenCV tracking.
\end{itemize}
